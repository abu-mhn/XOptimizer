What's New

Recent changes to X Optimizer.

Version v1.4.4

  • Profile photos & banners. Player photos and profile banners now show across tournaments — registrant list, match cards, standings, scoreboard and rankings.
  • Medal tags & themes. The ranking's top 3 get a Gold / Silver / Bronze Player badge — and each unlocks a matching reward theme.
  • Copy to share. The tournament Share button copies the message to your clipboard (a thumbs-up confirms) — paste it anywhere.
  • Tutorial on demand. The tutorial no longer pops up on its own — open it any time with the Tutorial button.
  • Skip re-registering. Sign in to a tournament you've already joined and you go straight in — no deck form again.
