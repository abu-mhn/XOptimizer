What's New

Recent changes to X Optimizer.

Version v1.5.8

  • Live achievement updates. The Achievement card now subscribes to Firebase in real time — counters tick up the moment a match credits, no page reload needed.
  • King of Jungle expanded. Need 1 Leon slot AND the other two slots each carry Rhino / Fox / Wolf / Viper / Tiger / Bear or Goat — the lion king and his pride.
  • Lonewolf tightened. Now also needs the Wolf slot's combo type (Attack / Defense / Stamina / Balance) to be the only one of its kind in the deck.
  • Sorcerer Supreme simplified. All three slots must contain a Wizard-named part (full wizard council).
  • Dragon roster grew. Dragon Tamer and Dragon Slayer now recognise Wyvern, Bahamut and Ragna in addition to Dran / Drake / Dragoon.
  • Auto-build button (Tester only). The Register popup has a new Auto-build (Test) button that fills the 3 slots with a deck satisfying any one of the seven achievements — for quick QA.
