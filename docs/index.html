What's New

Recent changes to X Optimizer.

Version v1.4.7

  • Pick any knockout size. Set the bracket size at create time — Top 4, 8, 16, 32, or a custom number. No more fixed Top 8.
  • Bulk Guests. One "+ Bulk Guests" button replaces "+ Register Others" — paste a name list (or just one) and every entry is added as a guest in one go.
  • Become Guest goes bulk too. Joining as guest opens the same bulk popup, so you can sign up friends alongside yourself in one tap before being dropped into the tournament.
