What's New

Recent changes to X Optimizer.

Version 2026-05-15

  • Co-hosts get full powers. Co-hosts can now start the tournament and edit participant names, just like the host.
  • Edit a registrant in place. Tap a registrant's name on the registering screen to change their name or rebuild their deck.
  • Tournament History shows all roles. Entries are now badged Host, Co-host, Participant, or Viewer — self-registered players are no longer lumped in with spectators.
  • Settings → Show Features. Open Settings and tap Show Features to see (and copy) a plaintext list of everything the app does.
