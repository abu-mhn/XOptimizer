What's New

Recent changes to X Optimizer.

Version v1.3.7

  • Sub-hosts by username. Hosts add co-hosts by username — no host code to share.
  • Round Robin fixes. A bye no longer counts as a win, and Top 8 works with odd-sized groups.
  • Revox results & history. Record a tournament result — the placing sets the points — and tap a member to see every event they joined. New "Revox Member" tag for view-only access.
  • Started tournaments stay viewable. A tournament stays in Open Tournaments after it starts, so anyone can still open and watch it.
