What's New

Recent changes to X Optimizer.

Version 2026-05-16

  • Rich share message. Tap Share in a tournament to fill in date, time, stadium, and rule. The message comes out formatted with a registration link at the bottom.
  • Every tournament is ranked. The host password is gone — every tournament you host now writes to the global ranking automatically.
  • Two new themes: Love and Forest. Settings → Theme adds a blush-pink and a woodland-green palette to the existing six.
  • Theme reaches further. Battle Pass, Parts Usage charts, the Show Features button, and the Deck name input now follow every theme — including Love and Forest.
