What's New

Recent changes to X Optimizer.

Version v1.5.7

  • Three new achievements. King of Jungle (Leon parts), Sharknado (Shark + Balance combo) and Sorcerer Supreme (Wizard slot flanked by two Attack-type slots). Each adds a title and theme at 100 wins.
  • Tighter achievement conditions. Dragon Slayer now also needs a Knight part in your own deck. Rush Hour now needs Clock Mirage + a Rush bit in the SAME slot.
  • Developer theme preview. Accounts tagged Developer can pick any locked theme (Revox / Medal / Achievement) without holding the matching tag.
  • X-Optimizer wordmark follows the theme. The header logo now adopts the active theme's text color — visible on every palette.
