What's New

Recent changes to X Optimizer.

Version v1.8.0

  • New blades + ratchet-less blade support. Fresh blades joined the roster — including Glory Valkyrie, Storm Pegasis and Heavens Ring. Blades flagged expandCX build with just a blade + bit (no ratchet, regular bit only), the same way Bullet Griffon works — the Calculator, Deck and Bey Check now force "No Ratchet" and limit the bit list automatically.
  • Participant limits. Set a maximum number of registrants when creating a tournament (with − / + steppers), or change it anytime from the registration screen. Registration closes automatically when full — the count shows N / cap with a red "Full" badge — and the Open Tournaments lobby shows each room's limit so players see it before joining.
  • Keeper role — fee tracking. A new "Keeper" tag, once invited to co-host a tournament, can tap each registrant to mark them Paid / Unpaid (the host can too). Everyone in the room sees a green "Paid ✓" badge and the count shows a paid tally.
  • Guest Judge role. A limited Judge who can be invited to co-host a tournament but can't create one of their own — handy for helpers who only judge.
  • Tidier registrant list on mobile. Each registrant's name and its Deck / Paid badges now stack into a clean two-line block on narrow screens instead of squeezing onto one row.
