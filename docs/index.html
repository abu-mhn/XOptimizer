Recent changes to X Optimizer.

Version v1.3.8

  • See a profile from a name. Hover or tap a username — in a tournament, the Revox list, or the Developer page — to view that player's profile card.
  • Edit Revox results. Fix or delete a member's recorded tournament results; their points re-total automatically.
  • Tag management. Developers can remove tags, not just add them, and tags are colour-coded by type.
  • Revox Member tag. A view-only Revox role — see the Revox tab and theme without editing rights.
