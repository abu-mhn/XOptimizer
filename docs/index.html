What's New

Recent changes to X Optimizer.

Version v1.5.5

  • Full deck required for signed-in registrations. Account registrations now have to fill every required part in every slot (blade + ratchet + bit, plus any mode-specific parts) — a blade-only or partially built slot is rejected with a clear "Slot N is missing parts" message. Guests are still exempt.
  • Confirm before removing a registrant. Tapping × next to a name now opens an "Are you sure you want to remove [name]?" prompt — an accidental tap can't boot a player by mistake any more.
  • Pre-Start deck check. Tapping Start Tournament now sweeps every registrant and surfaces deck problems in one combined prompt — names with no deck (account registrants only — guests are exempt), names with incomplete slots (and which slots), and names with banned parts (and which parts). The host can fix the listed issues or override and start anyway.
  • Bullet Griffon counts as complete without a ratchet. A slot with Bullet Griffon + a bit now passes the "Deck ✓" check (BG has a built-in ratchet, so the ratchet field is optional). Test-generated BG slots are also fixed to record this explicitly.
  • Match notifications reach everyone in the room. When a match goes LIVE, every connected device gets the toast and (if alerts are on) the system notification — host, co-host, participant and viewer alike. No more role gating.
  • Profile card placement fixed on mobile. Tapping a username on a phone no longer launches the profile dropdown into the top corner — it now clamps into the visible viewport near the tap instead of flying off to the edge of the screen.
