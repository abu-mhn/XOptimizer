What's New

Recent changes to X Optimizer.

Version v1.5.5

  • Full deck required for signed-in registrations. Every slot must have every required part. Guests still exempt.
  • Confirm before removing a registrant. Tapping × asks "Are you sure?" first.
  • Pre-Start deck check. Start Tournament lists every missing / incomplete / banned-parts deck in one prompt before starting.
  • Bullet Griffon counts as complete without a ratchet. BG's built-in ratchet means the ratchet field is optional.
  • Match notifications reach everyone in the room. Host, co-host, participant and viewer all get the toast.
  • Profile card placement fixed on mobile. Tapping a username no longer launches the card off-screen.
