What's New

Recent changes to X Optimizer.

Version v1.4.2

  • Tutorial. An 8-slide illustrated tutorial that auto-opens after 3 s idle on the Tournament tab — swipeable, with dots to jump.
  • Judge tag. Only Judge-tagged accounts can create tournaments, and the Sub-hosts suggestion list now lists only Judges.
  • Auto Revox ranking. RvX- / Rvx- usernames who finish Top 8 are entered into the Revox ranking automatically — 1st = 8 pts down to 8th = 1.
  • Single-line tag rows. Profile tags everywhere now fit on one line with an invisible horizontal scroller.
