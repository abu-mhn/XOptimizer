What's New

Recent changes to X Optimizer.

Version v1.4.6

  • Become Guest, no sign-in. Join a tournament as a guest with just a name — no account or anonymous sign-in needed.
  • Guests can skip the deck. Register a guest with no deck — empty decks are skipped from the finish pie chart.
  • Register Others for participants. Players in the tournament can sign up walk-in friends too, not just the host.
  • Locked name on edit. Re-opening an account registration keeps the name locked; guest entries stay editable.
  • Theme polish. Profile card, bio, deck edit, Developer page and part tiles now follow the active theme.
