What's New

Recent changes to X Optimizer.

Version v1.2.8

  • Host Test button. Hosts and co-hosts now have a "Test" button on the registering screen that bulk-adds synthetic participants (defaults to 10) with meta-random decks. Names auto-skip duplicates, and the writes go through in a single batched Firebase update.
  • No repeat parts in test decks. Each generated deck obeys the deck builder's "one of each part per deck" rule across all 3 slots. The exception (matching the regular Deck tab) is light lock chips — only the heaviest ones (Emperor / Valkyrie) are blocked from repeating.
  • Multi-mode part images. Parts with mode switching (Eclipse, Dual, Turbo, Operate, Scorpio Spear, Lightning L-Drago) now show their image everywhere on the dashboard. Previously their assets were stored per mode and the fallback URL pointed to a file that didn't exist.
  • Best Parts rank fix. If any one entry's image was missing, the whole row was hidden and 2nd/3rd appeared at positions 1st/2nd. Now only the broken image hides — the rank chip and part name stay in the correct slot.
  • Bey Check cleanup. Slot labels are now just "Slot 1 / 2 / 3" (no "· CX" / "· CX Expand" suffix), and the parts row's scrollbar is invisible — swipe still works.
  • Scoreboard theme polish. Advanced score buttons (Extreme / Burst / Over / Spin), the round counter, the corner ✕, and the centre divider all follow every theme now. The dark gash between the two sides is replaced with each theme's overlay background.
  • Tournament theming, round two. Registering view (heading / format pills / + Register myself / row names), group + match cards (Standings / Matches sub-tabs, round titles, match rows, seed pills, score cells), the Bey Check popup (player tabs + slot cards + labels), and the Rounds / Groups picker popups all now follow Light / Tropical / Love / Forest themes instead of staying dark.
