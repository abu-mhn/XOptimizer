What's New

Recent changes to X Optimizer.

Version v1.5.6

  • Achievement tab. Signed-in users get a new tab (next to Profile) tracking Dragon Tamer, Dragon Slayer, Lonewolf and Rush Hour — each unlocks a matching title and theme at 100 wins.
  • Bulk add participants. The + Add button now takes a name list (one per line) instead of one player at a time.
  • Remove participants mid-tournament. A new − Remove button drops players before round 2 starts.
  • Toolbar buttons show labels. Share, Sub-hosts, Add, Remove and Reset now read at a glance; the row scrolls sideways on narrow screens.
  • Tab icons follow the theme. Every tab icon (Dashboard, Calculator, Battle Pass, Buzz Bey, Developer, etc.) now adopts the active theme's text color.
