What's New

Recent changes to X Optimizer.

Version v1.4.3

  • Best Parts from history. Dashboard's Best Parts now mirrors the top entry in Tournament History — the last finished tournament, in any format.
  • Deck required. Registration blocks until all 3 deck slots are built; the error names the missing slot(s).
  • Register Others = guest. Players added via Register Others don't earn ranking points (no account attached); Register Myself still does.
  • Cleaner finish view. Once a tournament completes, the toolbar drops Share / Co-hosts / Add — only Close remains.
  • Clock Mirage Meta. Meta and Test now pick any -5 ratchet at random for Clock Mirage instead of preferring a non-existent "meta -5".
