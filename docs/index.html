What's New

Recent changes to X Optimizer.

Version v1.9.0

  • Battle Royale — new tab. Challenge another player to a points duel. You can only challenge someone with the same points or fewer; both stake an equal wager, you pick a Judge to oversee it, and the Judge declares the winner — who takes the pot. Battle Royale points are earned only by placing in tournaments, then gambled here. Opponents and Judges get notified.
  • Open or Closed tournaments. When you create a tournament you now choose its access. Open lists it publicly in the lobby; Closed keeps it private — it shows a "Closed" chip and asks for the room code when someone taps it. The host gets a copyable join code to share.
  • New blades + ratchet-less blade support. Fresh blades joined the roster — including Glory Valkyrie, Storm Pegasis and Heavens Ring. Blades flagged expandCX build with just a blade + bit (no ratchet, regular bit only), the same way Bullet Griffon works.
  • Participant limits + fee tracking + roles. Cap how many can register (auto-closes when full, shown in the lobby); a new Keeper role marks who's paid; and a Guest Judge role can co-host but not create tournaments.
