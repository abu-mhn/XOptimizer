What's New

Recent changes to X Optimizer.

Version v1.3.5

  • Labelled navigation. The tab bar now shows each tab's name under its icon, on wider tabs — no more guessing what an icon means.
  • Round Robin format. A new tournament format where everyone in a group plays everyone else once — pick it alongside Swiss and Single Elimination, with the same group-count and Top 8 options.
  • Your tournament, on any device. Sign in and you're dropped straight back into the tournament you host as host — on any phone or computer, no room codes to re-enter.
  • Add players mid-tournament. The new "Add Participant" button takes a name and a 3-combo deck — the player slots straight into the next round, no reset.
  • Rename anytime. Tap a player's name in a group's Standings to rename them, even after the tournament has started.
  • Adjust rounds on the fly. Tap "Round X / Y" in a group header to add or trim Swiss rounds mid-tournament — already-played rounds are kept.
  • Running tournaments stay in the lobby. Tournaments no longer disappear from the Open Tournaments list once they start — they show an "In progress" badge so co-hosts and viewers can still find them.
  • Cleaner rankings. Test registrants used for stress-testing no longer earn global ranking points or show up on the leaderboard.
  • Additional Data. Brachio, Whip, Outer and Narrow was added in library.
