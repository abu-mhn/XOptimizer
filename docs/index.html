What's New

  • New Deck tab — build a 3-combo deck, name it, shuffle the order, download the whole deck as a square PNG, or reset it. Tap a part image to open the same preview popup used in the library.
  • Result card has a new + cards button that adds the current combo to the first empty deck slot; a quick thumbs-up confirms the add.
  • Deck part-uniqueness rule: a part can only appear once across all three combos, except for lock chips — those can repeat, apart from Emperor and Valkyrie.
  • Downloaded result PNGs are now square (padded with the theme background), ready for Instagram sharing.
  • New Monochrome theme under Settings → Theme.
  • The header and favicon logos now switch cleanly between their light and dark variants to match the active theme.
  • Fixed 0-70's height data.
