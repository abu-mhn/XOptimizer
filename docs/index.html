What's New

Recent changes to X Optimizer.

Version v1.5.4

  • Edit your own deck. Participants can now tap their own row in the registrant list to re-open the edit popup — no need to ask the host.
  • Banned parts flagged on existing registrants. A red "Banned parts" badge replaces "Deck ✓" / "Incomplete" when a registrant's deck contains a banned part — catches the case where the host adds a ban after registration. The ban check is also cross-field now (a banned ratchet name flags the deck no matter which slot field it lives in).
  • Auto-kick on removal. When the host taps × to remove a registrant, that player's device auto-disconnects from the room with a clear notice — no stale sessions left open.
  • Phone-locked match notifications. A service worker is now registered so system match-start notifications fire on mobile Chrome / Edge / Brave even when the phone is locked or the tab is backgrounded. A new Match Alerts row in Settings shows the current state (off / blocked / on) and lets you turn alerts on or mute them per device.
  • Quieter notifications by role. Participants now only get notified for their own matches; viewers stay silent; host and co-hosts still get every match.
  • Banned Parts list scrolls horizontally. The Banned Parts panel keeps everything on one line with an invisible side-scroller, so a long ban list never pushes the page layout around.
