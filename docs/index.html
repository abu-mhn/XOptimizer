What's New

Recent changes to X Optimizer.

Version v1.5.10

  • Profile pane restore fix. Opening the Profile tab on a returning session now reliably loads your account card. It could previously stick on the "Sign in" prompt even when you were already signed in — auth detection now falls through onAuthChange, the Firebase listener and a short poll so the signed-in view appears whichever path resolves first.
  • Desktop tab bar centering. On desktop the top tab bar spans the full window and stays centered. Tabs that don't apply to you (Achievement before any unlock, Revox or Developer if you don't hold the tag) no longer shift the visible tabs off to one side.
