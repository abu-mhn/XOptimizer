What's New

Recent changes to X Optimizer.

Version v1.3.0

  • Simpler tournament setup. "Swiss" and "Swiss + Top 8" are now one option. Pick Swiss, then a popup asks whether to add a Top 8 knockout.
  • 3-group tournaments. Number of Groups now offers 2, 3, or 4. With Top 8, three groups seed top 2 per group plus the 2 best 3rd-place wildcards.
  • Host actions tidied. "+ Register myself" and "Test" now sit together in their own row under the Registrants heading.
