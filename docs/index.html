What's New

  • Swiss Tournament — now live across devices. Hitting Generate Groups now creates two shareable codes: a Co-host code for referees (view + score) and a Participant code for spectators (view only). Everyone's screen updates in real time.
  • LIVE toggle. Tap an unscored match to go live on it — a pulsing LIVE badge shows on every device so other refs don't double up. Tap the same card again to turn it off. One live match per device.
  • Tilt-to-score scoreboard. Scoring is mobile-only: tap a match, then rotate the phone to landscape to enter scores. The names/scores stay on screen after saving — the board only resets once you rotate back to portrait.
  • Top 8 knockout bracket. The instant every group's final round is saved, a single-elimination bracket auto-generates from the top 2 of each group — Quarterfinals → Semifinals → Final, plus full 3rd, 5th, and 7th place matches (QF losers drop into a consolation bracket).
  • Top 8 rankings. A ranked list above the bracket fills in as each placement match finishes, with gold / silver / bronze accents on the top three.
  • Better tie-breakers. Group standings now rank by Swiss points → Points Scored → Points Difference → Median-Buchholz.
  • Fix miscalculations anytime. Any completed match card can be tapped to re-open the scoreboard and correct the score — future rounds are not regenerated, so pairings stay stable.
