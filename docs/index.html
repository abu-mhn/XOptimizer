What's New

Recent changes to X Optimizer.

Version v1.7.0

  • Deck slots — card layout + stat graphs. Every combo is now its own card, with a collapsible Stat Graph (tap + / − to open) that draws a bar or radar chart following your Stat-display setting.
  • Switch a part's mode from the Deck. Multi-mode parts like Operate (Defense / Attack) show an inline mode chip on the slot — tap it to switch and the combo's image and stats recompute on the spot. The combo editor gained the same per-part mode control.
  • Ratchet-bit thumbnails fixed. Ratchet-bits such as Turbo and Operate now show their image in the Deck instead of a blank tile.
  • Mouse-wheel scrolling on desktop. The deck's part rows, toolbar and deck switcher now scroll sideways with a vertical mouse wheel — no trackpad swipe needed.
  • Tournament — set any placement depth. Single Elimination now lets you rank any number of finishers (not just Top 8 / 12 / 16), with a free-form picker that matches the Knockout Bracket Size layout and is editable from the registration screen.
  • Calculator result — fresh look. A rounded result card with framed stat, parts and data panels, a spec-sheet stats table, the part mode shown as a chip, and refined action buttons. Downloaded and shared photos render cleanly too.
